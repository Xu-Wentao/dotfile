- 使用百度千帆 AI 搜索进行网络搜索 - 执行实时网络搜索并获取相关内容
- 提供最新的信息和时事新闻数据
- 返回最相关网站的内容，包括标题、摘要、链接和权威度评分
- 支持按时间范围过滤（周/月/半年/年）
- 支持屏蔽特定网站
- 使用此工具获取超出知识库范围的信息
- 搜索在单次 API 调用中自动完成
- 并发控制: 同一轮中最多只能发起2个百度搜索请求

使用说明:
  - 必需参数: 查询内容(query)
  - 可选参数: 搜索版本(edition)、时间过滤器(search_recency_filter)、屏蔽网站列表(block_websites)
  - 返回结果包含来源网站、标题、摘要、链接、权威度评分和重排评分
  - 支持多种内容类型: 网页、图片、视频
  - 适用于查询最新信息、新闻、知识等
  - 如果需要多次搜索，每轮最多2个请求，超出必须等待上一次搜索完成

今天的日期是 {{date}}。在搜索最新信息或时事时，必须使用这一年
- 示例: 如果今天是 2025-07-15，用户询问"最新 AI 新闻"，应该搜索 "AI 新闻 2025"，而不是 "AI 新闻 2024"
