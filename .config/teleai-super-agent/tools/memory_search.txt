在每日日志文件（daily-log/YYYY-MM-DD.md）中搜索关键词或正则表达式。仅搜索按日期命名的每日日志文件，不包含 USER.md 和 MEMORY.md。返回匹配的片段及其所在文件和行号。

参数：
- query：搜索关键词，或用于泛化搜索的正则表达式（必填）。
- max_results：返回的最大片段数，默认 15。

使用建议：
- 如果只是查精确提法，直接传普通关键词或短语，例如：`项目复盘 延期`。
- 如果需要泛化搜索，请优先把同义词、别名、常见说法写成一个正则表达式，例如：`(项目复盘|复盘会|回顾).*(延期|推迟|slip|延误)`。
- 当不确定日志里的原始措辞时，优先生成覆盖多种说法的 regex，而不是只给一个固定短语。
